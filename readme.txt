Программа для выписывания рецептов.

Программа выписывания рецептов. Позволяет редактировать списки лекарств, болезней, симптомов, шаблонов рецептов. В тексте рецепта действует автозавершение фраз, что может значительно ускорить процедуру выписки.

Рецептор собирает список для автодополнения для файлов типа "Recept.html" из следующих источников:
1. Справочник "Лекарства".
2. Справочник "Симптомы".
3. Справочник "Заболевание".
4. Текстовый файл "Лекарства (текст)" (lek.txt)
5. Текстовый файл "Симптомы".(simp.txt)
6. Текстовый файл "Заболевание". (disease.txt)
Выборка происходит в момент создания файла "Recept.html".
